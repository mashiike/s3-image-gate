# s3-image-gate
upload image to Amazon S3 Bucket and check Content Moderation with Amazon Rekognition
## Usage

```
Usage of s3-image-gate
  -bucket string
        upload dest bucket (required)
  -debug
        enable debug log
  -help
        show help
  -index
        handle index
  -key-prefix string
        upload dest key prefix
  -port int
        http port (default 8000)
  -version
        show version
```
